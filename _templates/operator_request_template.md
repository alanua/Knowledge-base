# Operator Request Template

Use this format when starting work on any project.

## Project
Project name

## Objective
What must be achieved

## Context
Relevant business or technical context

## Constraints
- runtime constraints
- platform constraints
- security constraints
- cost or time constraints

## Current state
What already exists

## Expected output
- architecture decision
- Codex task
- review
- migration plan
- debugging plan

## Priority
critical / high / normal

## Notes
Any special instructions
